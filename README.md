# Sko4 API

Get all events
```
http://sko4.com/api/
```

Get event details
```
http://sko4.com/api/[id]
```

Get artist info
```
http://sko4.com/api/artist/[id]
```

Get venue info
```
http://sko4.com/api/venue/[id]
```
